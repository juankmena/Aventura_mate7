# Aventura Matemática 7°

App de práctica para el temario de Matemática 7°: conjuntos, números enteros, operaciones con enteros y potencias.

## Uso local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Subir a Streamlit Community Cloud

1. Crear un repositorio en GitHub.
2. Subir `app.py`, `requirements.txt` y esta carpeta `assets`.
3. Entrar a Streamlit Community Cloud.
4. Crear una nueva app conectada al repositorio.
5. Indicar como archivo principal: `app.py`.
6. Deploy.

La app quedará disponible en una URL tipo `https://nombre.streamlit.app` para abrir desde iPad, celular o computadora.
